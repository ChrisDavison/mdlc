# NAME

mdlc - markdown link checker

# SYNOPSIS

**mdlc [\-\-local] FILES...**

# DESCRIPTION

Check all the links for each file, or all under the current directory.

If *\-\-local* is passed, only check for links to other files, rather than web
links.
